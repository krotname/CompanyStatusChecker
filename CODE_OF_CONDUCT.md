# Code of Conduct

We value a welcoming and respectful environment for everyone.

This project is a public demonstration repository. When interacting in issues or pull
requests, contributors and users should:

- Be clear and professional in communication.
- Focus on the technical value of changes.
- Avoid harassment, abuse, or discriminatory language.
- Keep discussions on implementation details and project quality.

If a situation arises that harms constructive collaboration, we reserve the right to
remove comments or close discussions.
